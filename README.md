# Beyond RTL: Designing My First UART Controller in Verilog

> A Journey Through RTL Design, FPGA Verification, and ASIC GDSII Implementation

This repository contains my first complete digital design project: a UART controller built entirely from scratch in Verilog.

The project implements a programmable UART peripheral featuring a baud rate generator, UART transmitter, UART receiver, and a memory-mapped register interface. The design was verified through simulation, demonstrated on a Tang Nano 20K FPGA as a UART echo device, and successfully taken through the complete RTL-to-GDSII ASIC implementation flow using OpenLane and the SkyWater SKY130 process technology.

This project was built as a learning experience to strengthen my RTL design skills while gaining practical exposure to FPGA implementation and the digital ASIC design flow.
